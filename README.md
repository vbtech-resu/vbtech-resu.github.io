# VBTech Resume Builder

A frontend-only resume builder that allows users to create
professional resumes and download them as PDF.

## Features
- Live resume preview
- Photo upload
- Single / Two column layout
- Education & experience sections
- PDF download (A4 size)
- Fully frontend (HTML, CSS, JavaScript)

## Live Demo
https://vbtech-resu.github.io/

## Usage
This tool is free to use for creating resumes.
